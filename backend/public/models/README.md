# 3D-модели Urban Sprint

Положите сюда файлы:

- `urban-sprint-night-pulse.glb` / `.usdz`
- `urban-sprint-midnight-gold.glb` / `.usdz`
- `urban-sprint-arctic-drive.glb` / `.usdz`
- `urban-sprint-stealth-mode.glb` / `.usdz`

Пока файлов нет, API отдаёт демо-модель с modelviewer.dev.

Загрузка через админку: **AR-модели → Загрузить .glb / .usdz**
