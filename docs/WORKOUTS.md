# Тренировки и GPS

## Жизненный цикл

1. **Старт** — `POST /api/workouts/start` (нужны активные кроссовки).
2. **Трекинг** — GPS + шаги, локальное сохранение + sync на сервер.
3. **Завершение** — удержание кнопки «Стоп» 2 сек → `POST /api/workouts/:id/finish`.
4. **Валидация** — сервер проверяет дистанцию, время, GPS, лимиты.
5. **Бонус** — списание с фонда, зачисление клиенту.

## Правила начисления бонусов

| Параметр | По умолчанию |
|----------|--------------|
| Бонус за 1 км | 3 сомони |
| Дневной лимит | 10 сомони |
| Лимит на пару кроссовок | 200 сомони |
| Мин. дистанция | 500 м |
| Мин. время | 5 минут |

Настраивается в админке → **Настройки бонусов**.

## Фильтрация GPS (клиент)

Файл: `mobile/src/services/gpsFilter.js`

Перед записью точка проходит фильтры:

| Фильтр | Условие |
|--------|---------|
| Точность | ≤ 15 м (первая точка ≤ 80 м) |
| Интервал | 1–15 сек между точками |
| Смещение | ≥ 2.5 м от предыдущей |
| Скорость | < 0.2 м/с → не записывать сегмент |
| Скачок | > 80 м за < 4 сек → отбросить |

### Расчёт метрик

- **Дистанция** — сумма Haversine между отфильтрованными точками.
- **Текущая скорость** — скользящее среднее по 3 точкам, м/с → км/ч.
- **Средняя скорость** — `дистанция (км) / время в движении (ч)`.
- Время на паузе (ручной и автопаузе) **не учитывается**.

### Автопауза

- Скорость < **0.5 м/с** в течение **4 сек** → автопауза.
- Возобновление при скорости > **0.8 м/с**.
- Grace period **45 сек** после старта, пока нет точек маршрута.

## Синхронизация с сервером

- Точки отправляются каждые **4 сек** (`POST /api/workouts/:id/points`).
- Синхронизация работает **даже на паузе** (для Live GPS в админке).
- Сервер фильтрует дубли (`backend/src/utils/geo.js`).

## Live GPS в админке

`GET /api/admin/workouts/live` — все тренировки `in_progress` с точками.

Обновление карты: каждые 4 сек (polling).

## Тесты приёмки (QA)

| Тест | Ожидание |
|------|----------|
| Телефон на столе 5 мин | Дистанция 0, скорость 0 |
| Бег по прямой | Плавный трек без «паутины» |
| Остановка 1 мин | Средняя скорость не падает сильно (автопауза) |
