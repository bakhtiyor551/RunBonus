# Админ-панель

SPA на **React + Vite**. Dev: `http://localhost:5174`.

Вход после seed: **admin** / **admin123**.

## Разделы

| Раздел | Функции |
|--------|---------|
| **Dashboard** | Сводная аналитика, KPI |
| **Клиенты** | Список, профиль, баланс, блокировка, сброс устройства |
| **QR / Кроссовки** | Генерация партий, статусы активации |
| **Тренировки** | История, **Live GPS**, детали маршрута |
| **Бонусные счета** | Фонд компании, пополнение, транзакции |
| **Настройки бонусов** | Цена за км, лимиты |
| **Уровни клиентов** | Прогрессия по километражу |
| **Списание бонусов** | Ручные операции |
| **Вывод средств** | Заявки, одобрение/отклонение |
| **Магазин** | Товары, категории, цвета |
| **Склад** | Остатки, движения |
| **Заказы** | Обработка заказов, курьеры |
| **Способы оплаты** | Методы, доставка, кошельки |
| **Отчёты** | Продажи, тренировки, финансы |
| **Реклама** | Кампании, баннеры, push |

## Live GPS

Когда клиент начинает тренировку в приложении:

1. Откройте **Тренировки**.
2. Вверху появится блок **Live GPS** (если есть активные тренировки).
3. Карта обновляется каждые **4 секунды**.
4. Клик по клиенту → детали с картой и таблицей точек.

API: `GET /api/admin/workouts/live`

Требования:

- Клиент отправляет GPS на сервер (`POST /api/workouts/:id/points`).
- Интернет у клиента.
- Backend запущен и доступен.

## Управление бонусным фондом

1. **Бонусные счета** → откройте счёт типа `bonus_fund`.
2. **Пополнить** — зачисление на фонд.
3. При одобрении тренировки бонус списывается с фонда и зачисляется клиенту.
4. Если фонд пуст — статус `rejected_no_fund`.

## Генерация кроссовок

1. **QR / Кроссовки** → **Сгенерировать партию**.
2. Укажите модель и количество.
3. Распечатайте QR-коды для маркировки обуви.

## Сборка

```bash
cd admin
npm install
npm run build
```

Статика в `admin/dist/` — раздавать через nginx или вместе с backend.

## API-клиент

`admin/src/api.js` — обёртка над `fetch` с JWT из `localStorage`.

```javascript
import { adminApi } from './api';
const users = await adminApi('/api/admin/users');
```
