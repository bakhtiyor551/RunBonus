# Urban Sprint — фото для 360°

| Файл | Назначение |
|------|------------|
| `hero-360.png` | Исходный макет (полный экран) |
| `view-main.png` | Крупный ракурс (авто-вырезка) |
| `angle-left.png` … `angle-sole.png` | 6 ракурсов (авто-вырезка из макета) |

Чтобы обновить ракурсы: замените PNG или положите свои файлы с теми же именами, затем при необходимости:

```bash
node scripts/crop-urban-angles.mjs
```

Замените PNG своими фото тех же ракурсов для лучшего качества 360°.
